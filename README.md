# 🌤️ Real-Time Weather Analytics Dashboard

## 📋 **Project Overview**

An advanced **interactive weather analytics dashboard** built using Microsoft Power BI, featuring real-time data integration through weather APIs. This comprehensive business intelligence solution provides detailed meteorological insights across multiple Indian cities, demonstrating expertise in data visualization, API consumption, and dashboard development.

## 🎯 **Key Features**

### 🔄 **Real-Time Data Integration**
- Live weather data fetching through API connections
- Automatic data refresh capabilities
- Dynamic city selection functionality

### 📊 **Comprehensive Weather Metrics**
- **Temperature Analytics**: Current temperature, feels-like temperature, and forecasting
- **Atmospheric Conditions**: Humidity levels, barometric pressure, wind speed
- **Environmental Monitoring**: UV Index, precipitation levels
- **Air Quality Assessment**: PM10, PM2.5, O3, CO, NO2 pollutant tracking
- **Astronomical Data**: Sunrise and sunset times
- **Precipitation Forecasting**: Rain probability percentages

### 🏙️ **Multi-City Coverage**
Primary focus on **5 strategic Indian locations**:
- **Shimla** (Hill Station - Himachal Pradesh)
- **Gorakhpur** (Uttar Pradesh)
- **Jamshedpur** (Industrial City - Jharkhand)  
- **Leh** (High Altitude - Ladakh)
- **Siswa Bazar** (Local Region)

### 📈 **Interactive Visualizations**
- **Temperature trend graphs** with 3-day forecasting
- **Air Quality Index** with pollutant breakdown (donut charts)
- **Weather parameter cards** with real-time values
- **Precipitation probability** bar charts
- **Responsive design** with city switching capabilities

## 🛠️ **Technical Implementation**
